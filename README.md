## Deployment

### Vercel

This site is ready to deploy on Vercel. Follow these steps:

1. **Push to GitHub**
   ```bash
   git add .
   git commit -m "Initial commit: Satyam Sharma Portfolio"
   git branch -M main
   git remote add origin https://github.com/yourusername/satyam-sharma-portfolio
   git push -u origin main
   ```

2. **Deploy on Vercel**
   - Go to [vercel.com](https://vercel.com)
   - Sign up or log in with your account
   - Click "New Project"
   - Import your GitHub repository
   - Vercel will auto-detect settings from `vercel.json`
   - Click "Deploy"

3. **Custom Domain** (Optional)
   - In Vercel dashboard, go to Settings > Domains
   - Add your custom domain
   - Update DNS records as instructed

## Customization

### Update Social Links
Edit the contact section in `index.html`:
- Replace email address
- Update LinkedIn URL
- Add GitHub profile
- Add Twitter handle

### Change Colors
Modify CSS variables in `style.css`:
```css
:root {
    --primary-color: #00d4ff;      /* Cyan */
    --secondary-color: #ff006e;    /* Pink */
    --accent-color: #00ff88;       /* Green */
    /* ... other variables ... */
}
```

### Update Content
- Hero section: Update heading and description
- Experience: Add detailed experience entries
- Skills: Add more skill categories
- Achievements: Update achievement cards
- Education: Modify degree and university info
